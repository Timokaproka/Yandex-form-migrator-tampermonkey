# Yandex Forms Migrator & Ultimate Manager

Профессиональный набор инструментов для автоматизации, массового управления и расширенной аналитики в сервисе Яндекс Формы. Проект позволяет обходить ограничения стандартного интерфейса, экспортировать/импортировать формы в JSON и эффективно интегрировать контент, сгенерированный ИИ.

## 🚀 Основные возможности

* **Массовый экспорт/импорт:** Перенос десятков форм и целых папок между аккаунтами (личными и бизнес-профилями 360/Cloud) в один клик.
* **AI-Интеграция:** Поддержка конвейера «Текст теста -> AI (JSON) -> Препроцессор -> Яндекс Форма».
* **Управление структурой:** Массовое удаление и экспорт групп с автоматическим восстановлением вложенности папок.
* **Smart Resolve:** Интеллектуальные алгоритмы обработки навигации и кэширования элементов интерфейса Яндекса.

## 🛠 Состав инструментов

### 1. [Yandex Forms Ultimate Manager](https://github.com/Timokaproka/Yandex-form-migrator-tampermonkey/raw/refs/heads/main/Yandex%20Forms%20Ultimate%20Manager%20(V10.17.1).user.js)

Флагманский инструмент для управления всеми вашими формами и папками на главной странице сервиса. Добавляет возможность массового выделения элементов для их последующего переноса или удаления.

> **Рекомендация:** Для максимально комфортной работы и полного контроля над данными крайне рекомендуется использовать данный менеджер совместно с **Migrator Single**.

### 2. [Yandex Forms Migrator Single](https://github.com/Timokaproka/Yandex-form-migrator-tampermonkey/blob/main/Yandex%20Forms%20Migrator%20Single%20(V9.11).user.js)

Специализированный инструмент для работы с одиночными формами непосредственно в режиме редактора. Позволяет мгновенно выгрузить текущую форму в JSON или загрузить новую структуру в уже открытый черновик.

### 3. [Yandex Forms JSON Preprocessor](https://github.com/Timokaproka/Yandex-form-migrator-tampermonkey/blob/main/Yandex%20Forms%20JSON%20Preprocessor%20(V1.5).user.js)

Технологический мост между ИИ и системой Яндекса. Конвертирует упрощенный JSON, созданный нейросетью, в полноценный системный формат с генерацией необходимых технических ключей и валидацией данных.

### 4. [Yandex Forms Image Uploader](https://github.com/Timokaproka/Yandex-form-migrator-tampermonkey/blob/main/Yandex%20Forms%20Image%20Uploader%20(V1.1).user.js)

Вспомогательная утилита для оптимизации работы с медиаконтентом.

* **Для чего он нужен:** Это инструмент для быстрой подготовки и пакетного размещения ваших изображений на серверах Яндекса. Вместо того чтобы вручную прикреплять картинку к каждому вопросу через конструктор, вы подготавливаете все иллюстрации заранее. Это делает процесс создания тестов с визуальными материалами значительно быстрее и удобнее, особенно при автоматизированном импорте.
* **Как это работает:** Скрипт добавляет интерфейс для выбора группы фотографий с вашего компьютера. Он автоматически загружает их на облачную инфраструктуру Яндекса и предоставляет вам готовый список прямых ссылок. Эти ссылки можно сразу использовать в JSON-файлах или передавать ИИ — система Яндекса мгновенно распознает их, и ваши изображения будут корректно отображаться в формах.

## 📦 Установка

1. Установите расширение **Tampermonkey** в ваш браузер.
2. Выберите необходимый скрипт из списка выше.
3. Нажмите кнопку установки в Tampermonkey (скрипт подхватится автоматически при открытии ссылки на `.user.js` файл).
4. Перейдите на страницу `forms.yandex.ru` — новые функции активируются автоматически.

## 🔄 Рабочий процесс (Workflow)

1. **AI:** Генерируете структуру теста с помощью нейросети по шаблону из документации.
2. **Preprocessor:** Пропускаете полученный текст через конвертер для получения полного технического файла.
3. **Manager/Single:** Используете функцию «Импорт» на главной странице или в редакторе для создания готовой формы.
