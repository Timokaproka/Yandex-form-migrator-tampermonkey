# Yandex Forms Migrator & Ultimate Manager

Набор профессиональных пользовательских скриптов для автоматизации работы с сервисом Яндекс Формы. Проект расширяет стандартный функционал платформы, позволяя массово управлять формами, создавать резервные копии, переносить данные между личными и корпоративными аккаунтами (Yandex 360/Cloud), а также интегрировать формы, созданные с помощью искусственного интеллекта.

## Основные возможности

* **Массовое управление:** Экспорт, импорт и удаление десятков форм в один клик.
* **Работа с папками:** Сохранение и перенос целых групп форм с автоматическим восстановлением структуры каталогов на новом аккаунте.
* **Интеграция с нейросетями:** Преобразование текстовых тестов, сгенерированных ИИ, в готовые интерактивные формы.
* **Независимость от ограничений:** Обход системных лимитов интерфейса Яндекса на пакетную обработку и работу со сторонними медиафайлами.

## Состав комплекса инструментов

Для полноценной работы инструменты разделены на четыре независимых модуля. Вы можете устанавливать только те, которые вам необходимы.

### 1. Yandex Forms Ultimate Manager

Главный скрипт для управления формами на стартовой странице сервиса. Добавляет систему чекбоксов (галочек) рядом с каждой формой и папкой. Позволяет выделить нужные элементы и применить к ним массовые действия: скачать архивом, загрузить новые или удалить.
*Примечание: для максимально комфортной работы рекомендуется использовать этот скрипт в связке с Yandex Forms Migrator Single.*

### 2. Yandex Forms Migrator Single

Скрипт для точечной работы внутри редактора конкретной формы. Добавляет функции импорта и экспорта непосредственно в меню настроек формы. Позволяет быстро выгрузить или заменить структуру текущей формы, не возвращаясь на главную страницу.

### 3. Yandex Forms JSON Preprocessor

Конвертер данных для работы с нейросетями. Текстовые ИИ-модели не способны сразу выдать сложный системный код, который требует Яндекс. Этот скрипт берет упрощенный текстовый файл от нейросети и автоматически достраивает всю необходимую техническую архитектуру, превращая его в полноценную форму, готовую к импорту.

### 4. Yandex Forms Image Uploader

Специализированная утилита для решения проблемы с изображениями при автоматическом создании форм.

**Для чего он нужен:** Это инструмент для быстрой подготовки и пакетного размещения ваших изображений на серверах Яндекса. Вместо того чтобы вручную прикреплять картинку к каждому вопросу через конструктор, вы подготавливаете все иллюстрации заранее. Это делает процесс создания тестов с визуальными материалами значительно быстрее и удобнее, особенно при автоматизированном импорте.

**Как это работает:** Скрипт добавляет интерфейс для выбора группы фотографий с вашего компьютера. Он автоматически загружает их на облачную инфраструктуру Яндекса и предоставляет вам готовый список прямых ссылок. Эти ссылки можно сразу использовать в JSON-файлах или передавать ИИ — система Яндекса мгновенно распознает их, и ваши изображения будут корректно отображаться в формах.

## Инструкция по установке

Процесс установки не требует навыков программирования и занимает несколько минут:

1. Установите расширение **Tampermonkey** для вашего браузера (доступно в официальных магазинах расширений Chrome, Edge, Firefox, Яндекс Браузер).
2. Перейдите на страницу нужного скрипта в этом репозитории (файлы с расширением `.user.js`).
3. Нажмите кнопку **Raw** (или просто скопируйте код скрипта) — расширение Tampermonkey автоматически распознает его и предложит нажать кнопку **"Установить"**.
4. Откройте страницу Яндекс Форм. Обновите страницу, если она была открыта. Новые функции, кнопки и меню появятся в интерфейсе автоматически.

## Сценарий работы с нейросетями (AI Workflow)

Если ваша цель — быстро создавать тесты или опросы с помощью ИИ:

1. Отправьте нейросети текст вашего теста и попросите структурировать его согласно правилам из файла документации к упрощенному JSON.
2. Загрузите полученный от ИИ файл в **JSON Preprocessor** (кнопка появится в углу экрана на странице Яндекс Форм).
3. Полученный полноценный файл импортируйте через **Ultimate Manager** (на главной странице) или **Migrator Single** (в редакторе формы).

## Лицензия и условия использования

Проект предоставляется "как есть" и распространяется для личного и образовательного использования. Инструменты работают на базе текущей архитектуры интерфейса Яндекс Форм.
